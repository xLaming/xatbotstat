# BOTSTAT

This is a tool for xat users, with it you can do a lot of external tools for you own usage, actually there is some like showing the current time in your PC, current date etc.

# Available commands:
* date
* clock
* randomavatar
* setname
* setavatar
* setstatus
* setgame

# Required:
* Power BOTSTATS and STATUS
